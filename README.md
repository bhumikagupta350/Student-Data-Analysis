# Student Data Analysis System (Python Project)

##  Overview

This project is a **Student Data Analysis System** built using Python.
It reads student data from a CSV file, performs analysis, assigns grades, ranks students, and visualizes performance using graphs.

This project demonstrates **data handling, logic building, and basic data visualization**, making it a strong beginner-to-intermediate level project.

---

##  Features

* 📥 Read student data from CSV file
* 📊 Calculate:

  * Average marks
  * Topper & lowest scorer
  * Score gap
*  Automatic grade assignment (A, B, C, F)
*  Student ranking system
*  Search student by name
*  Save processed data to new CSV
*  Bar chart visualization:

  * Highlights topper (green)
  * Highlights failed students (red)
  * Shows average line
*  Pie chart visualization:

  * Displays grade distribution
